# MEDIA

This folder contains visual assets used in the project:  
diagrams, charts, concept maps, graphics, or safely shareable imagery.

## Includes
- diagrams  
- charts  
- conceptual maps  
- GitHub-safe images  
- explanatory visuals  

## Does Not Include
- photos of yourself  
- identifiable location imagery  
- anything tied to private investigation or personal life  

## Naming Convention
media-topic.ext
